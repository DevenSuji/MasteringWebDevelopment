The HTML BOLD element defines bold text, without any extra importance.


The HTML STRONG element defines text with strong importance. The content inside is typically displayed in bold.

The HTML ITALIC element defines a part of text in an alternate voice or mood. The content inside is typically displayed in italic.

The HTML Emphasized Text element defines emphasized text. The content inside is typically displayed in italic.

The HTML Small Text element defines smaller text

The HTML Inserted Text element defines a text that has been inserted into a document. Browsers will usually underline inserted text

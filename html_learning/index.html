HTML Tags Quick Reference

A basic list of common HTML tags and simple code examples.

Note: This list covers many common tags but is not absolutely exhaustive. It demonstrates basic usage; tags often have many more attributes and complex uses.

<h1> to <h6> - Headings

<h1>Level 1 Heading</h1>
<h2>Level 2 Heading</h2>
<h3>Level 3 Heading</h3>
<h4>Level 4 Heading</h4>
<h5>Level 5 Heading</h5>
<h6>Level 6 Heading</h6>

<p> - Paragraph

<p>This is a paragraph of text. It can contain multiple sentences.</p>
<p>This is another paragraph.</p>

<br> - Line Break

This text is on one line.<br>This text is on the next line.

<hr> - Horizontal Rule

<p>Some content above the line.</p>
<hr>
<p>Some content below the line.</p>

<pre> - Preformatted Text

<pre>
  This text preserves
  whitespace and line breaks
  exactly as written.
</pre>

<blockquote> - Block Quotation

<blockquote cite="http://example.com">
  This is a long quotation indented from the surrounding text.
</blockquote>

<a> - Anchor (Hyperlink)

<a href="https://www.google.com" target="_blank">Visit Google</a>

<strong> & <b> - Importance / Bring Attention

<p>This text is <strong>very important</strong>.</p>
<p>This text is <b>bolded</b> for attention.</p>

<em> & <i> - Emphasis / Idiomatic Text

<p>You <em>must</em> see this.</p>
<p>The term <i>HTML</i> stands for HyperText Markup Language.</p>

<code> - Code Fragment

<p>Use the <code>console.log()</code> function.</p>

<span> - Generic Inline Container

<p>Part of this text is <span style="color: blue;">blue</span>.</p>

<sub> & <sup> - Subscript & Superscript

<p>Water is H<sub>2</sub>O.</p>
<p>E = mc<sup>2</sup>.</p>

<ul> & <li> - Unordered List

<ul>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ul>

<ol> & <li> - Ordered List

<ol>
  <li>Step one</li>
  <li>Step two</li>
  <li>Step three</li>
</ol>

<dl>, <dt>, <dd> - Description List

<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
</dl>

<img> - Image

<img src="placeholder.jpg" alt="A placeholder image" width="100" height="50">
<!-- Replace 'placeholder.jpg' with a real image URL -->

<audio> - Audio

<audio controls src="audiofile.mp3">
  Your browser does not support the audio element.
</audio>
<!-- Replace 'audiofile.mp3' with a real audio file URL -->

<video> - Video

<video controls width="300" src="videofile.mp4">
  Your browser does not support the video element.
</video>
<!-- Replace 'videofile.mp4' with a real video file URL -->

<table>, <tr>, <th>, <td> - Table

<table border="1">
  <caption>Simple Data Table</caption>
  <thead>
    <tr>
      <th>Header 1</th>
      <th>Header 2</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Row 1, Cell 1</td>
      <td>Row 1, Cell 2</td>
    </tr>
    <tr>
      <td>Row 2, Cell 1</td>
      <td>Row 2, Cell 2</td>
    </tr>
  </tbody>
</table>

<form> & Basic Inputs

<form action="/submit-page" method="post">
  <fieldset>
    <legend>User Info:</legend>

    <label for="username">Username:</label><br>
    <input type="text" id="username" name="username" required><br><br>

    <label for="password">Password:</label><br>
    <input type="password" id="password" name="password" required><br><br>

    <label for="bio">Bio:</label><br>
    <textarea id="bio" name="bio" rows="4" cols="30"></textarea><br><br>

    <label for="color">Favorite Color:</label>
    <select id="color" name="color">
      <option value="red">Red</option>
      <option value="blue">Blue</option>
      <option value="green">Green</option>
    </select><br><br>

    <input type="checkbox" id="subscribe" name="subscribe" value="yes">
    <label for="subscribe">Subscribe to newsletter</label><br><br>

    <input type="radio" id="option1" name="choice" value="opt1">
    <label for="option1">Option 1</label><br>
    <input type="radio" id="option2" name="choice" value="opt2">
    <label for="option2">Option 2</label><br><br>

    <button type="submit">Submit</button>
    <input type="reset" value="Reset Form">
  </fieldset>
</form>

<div> - Generic Block Container

<div style="border: 1px solid red; padding: 10px;">
  This is a div element, often used for layout and styling.
</div>

Semantic Structure: <header>, <nav>, <main>, <section>, <article>, <aside>, <footer>

<body> <!-- Styles in head make borders visible -->

  <header>
    <h1>Website Title</h1>
    <nav>
      <ul>
        <li><a href="#">Home</a></li>
        <li><a href="#">About</a></li>
      </ul>
    </nav>
  </header>

  <main>
    <section>
      <h2>Main Content Section</h2>
      <article>
        <h3>Article Title</h3>
        <p>Article content goes here...</p>
      </article>
      <article>
        <h3>Another Article</h3>
        <p>More content...</p>
      </article>
    </section>

    <aside>
      <h3>Sidebar</h3>
      <p>Related links or ads.</p>
    </aside>
  </main>

  <footer>
    <p>&copy; 2025 Website Name</p>
  </footer>

</body>

<details> & <summary> - Disclosure Widget

<details>
  <summary>Click to see details</summary>
  <p>Here are the hidden details that are revealed when you click.</p>
</details>

Metadata Examples (Typically in <head>)

<!-- Sets character encoding -->
<meta charset="UTF-8">

<!-- Sets viewport for responsive design -->
<meta name="viewport" content="width=device-width, initial-scale=1.0">

<!-- Page description for SEO -->
<meta name="description" content="A brief description of the page.">

<!-- Link to external CSS file -->
<link rel="stylesheet" href="styles.css">

<!-- Page Title (in browser tab) -->
<title>My Web Page</title>

<!-- Inline CSS -->
<style>
  p { color: navy; }
</style>

<!-- Link to external JavaScript file -->
<script src="script.js" defer></script>
